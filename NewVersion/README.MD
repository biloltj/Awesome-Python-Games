## ğŸš€ Getting Started

To play these games on your local machine, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/biloltj/Awesome-Python-Games.git](https://github.com/biloltj/Awesome-Python-Games.git)
   cd Awesome-Python-Games
   ```
2. **Create a Virtual Environment (Recommended):**
```bash
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate
```
3. **Install Dependencies:**
```bash
pip install -r requirements.txt
```
